## Features

 - **Allow users to view other players' gacha (headhunting) history by entering their UID**
 
 - **Implements filtering for gacha records based on user-specified date ranges**
 
 - **Shows total gacha count: `Gacha time: X (unfiltered)` or `Gacha time: X/Y (filtered/total)`**
